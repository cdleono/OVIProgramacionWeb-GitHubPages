Eliminación de Registros

Contenido de la Sección

Especificar el paso a paso de la eliminación de un registro en la base de dato que se va mostrar en la página web desde PHP.

1. Conectar la base de datos por medio de la variable $link.

2. Crear una variable ($q) que almacene la eliminación del registro.

3. Crear la sentencia con un DELETE..

4. Crear una variable ($rs) que almacene la eliminación del registro con el mysql_query.

5. Aplicar la sentencia if para que si la dicho en la variable $rs se cumple o no se cumple, imprima un mensaje indicando lo sucedido.

6. Desconetar la base de datos por medio de la variable $link.
