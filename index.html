Objetivo General

En este objeto virtual de aprendizaje (OVA) se mostrará la vinculación de una bases de datos a una página web con el lenguaje de programación PHP, creando opciones de consulta, edición, modificación y elemininación de datos y/o registros.

Objetivos Específicos

1. Mostrar la programación necesaria para lograr la vinculación.

2. Explicar el paso a paso de la creación de las opciones de consulta y edición de registros desde PHP.

3. Explicar el paso a paso de la creación de las opciones de modificación y eliminación desde PHP.
