Consulta de Registros

Contenido de la Sección

Especificar el paso a paso de la creacion de una consulta a la bases de datos que se va mostrar en la página web desde PHP.

1. Conectar la base de datos por medio de una variable.

2. Crear una variable que almacene la consulta ("SELECT * FROM TABLA").

3. Crear una variable que almacene el resultado de ejecutar la consulta.

4. Crear una variable que almacene el recorrido por las filas de la tabla.

5. Añadir un While que recorre todas las filas de la tabla que se esta consultando mientras se cumpla la condición.

6. Insertar un echo el cual imprime y muestra contenido html dentro de PHP.
